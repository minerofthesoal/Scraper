Features

📄

Universal Text Extraction

Scrapes text from any site including SPAs, shadow DOM, web components, and dynamically-loaded content with multi-strategy fallback.

📸

Image & Media Scraping

Extracts images (including lazy-loaded, CSS backgrounds, srcset), audio sources, and video files with YouTube filtering toggle.

🔗

Link Discovery

Finds all links including onclick navigation, data-href attributes, role="link" elements, and JavaScript-rendered URLs.

🤖

AI-Powered Extraction

Uses NuExtract-2.0-2B for intelligent structured data extraction. Supports custom templates, local regex fallback, and combined "All" mode.

📊

GwSS Visualization

Interactive force-directed graph showing all scraped sites with live physics, favicons, unique composite edge patterns, and per-site data flow diagrams (SSDg).

🚀

HuggingFace Upload

One-click upload to HuggingFace datasets with automatic JSONL sharding, MLA/APA citations, and community dataset support.

⚡

Batch Queue & Auto-Scan

Queue multiple URLs for background scraping. Auto-scan crawls linked pages automatically with configurable rate limiting and domain filtering.

🛡

Privacy & Security

Content sanitization, robots.txt respect, sensitive data filtering (PII, API keys, slurs), and configurable redaction modes.

💻

Python CLI

Full-featured command-line interface with 55+ commands for data management, AI model serving, HuggingFace operations, and export formatting.

Installation

  1. Download the latest .xpi file from the Releases page
  2. Open Firefox and navigate to about:addons
  3. Click the gear icon and select "Install Add-on From File..."
  4. Select the .xpi file and approve the permissions
  5. Optional: Install the CLI with python install.py for additional features

CLI Quick Start

# Install the CLI tool
python install.py

# View all commands
scrape help

# Export scraped data
scrape export --format jsonl

# Start AI extraction server
scrape ai.serve

# Upload to HuggingFace
scrape hf.push

# View scraping statistics
scrape stats
