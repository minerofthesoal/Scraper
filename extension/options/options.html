Uses NuExtract-2.0-2B for intelligent structured data extraction. When enabled, the CLI will automatically download and cache the NuExtract model locally. No separate server process required. Disable this if you want to run your own server instead.
Only needed if auto-download is off. Start with: scrape ai.serve
Hardware requirements:
GPU: NVIDIA GTX 1070 (8GB+) — recommended, ~2s per extraction
CPU: Any modern processor — works but slower, ~15-30s per extraction
Manual setup: pip install torch transformers then scrape ai.setup
